Design System

TRDS is a vanilla javascript component library and design system made for our company's online presence. Compatible with React.

Theming

The library has got a theme.css file where you can set the fonts and colors for theming using css variables. Use this css file with a link tag in head. Keep in mind you have to provide the fonts path in this file.

Typhography

Elements to use: span, p, h1...h6
Classes to use: size-{xs...xxl}, variant--1
